## 🚀 Quick Deploy Commands

### Build
```bash
npm run build
```

### Test Production Build
```bash
npm run preview
```

### Deploy to Vercel (CLI)
```bash
npm i -g vercel
vercel
```

### Deploy to Netlify (CLI)
```bash
npm i -g netlify-cli
netlify deploy --prod
```
