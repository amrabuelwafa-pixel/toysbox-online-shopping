---

## 🚀 Quick Start (3 Steps)

### Step 1: Change Password
Open `public/auth.html` and change line 127:
```javascript
const correctPassword = 'YOUR_PASSWORD_HERE';
```

### Step 2: Deploy
Choose one:
- **Vercel**: Push to GitHub → Import on vercel.com
- **Netlify**: Push to GitHub → Import on netlify.com  
- **Local**: Run `npm run preview`

### Step 3: Share
Share your URL and password with authorized users!
